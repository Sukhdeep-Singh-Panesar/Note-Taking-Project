# Note Taking Project

## Description
This project is a note-taking application that allows users to create, manage, and organize notes. It is built as a practice project to demonstrate development skills and application structure.

## Features
- Create notes
- Edit notes
- Delete notes
- View saved notes


## Installation


## Installation

1. Clone the repository:

- git clone https://github.com/Sukhdeep-Singh-Panesar/Note-Taking-Project.git


2. Navigate into the project directory:

- cd Note-Taking-Project


3. Install dependencies:

- npm install


## Running the Project

Start the server:

- node server.js


## Usage
After starting the server, open your browser and go to:

- http://localhost:3000
- Then start creating and managing your notes.
